# MurMur Codex Factory

Monorepo scaffold for MurMur OS dashboard, API, agent system, shared packages, and project workspaces.

## Structure

- `apps/` runtime applications (`dashboard`, `api`, `agents`)
- `projects/` project-specific execution tracks
- `packages/` shared internal libraries
- `supabase/` SQL schema and seed data
- `codex/` prompts, task definitions, and automation pipelines
