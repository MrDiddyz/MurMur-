# MurMur AI

MurMur is an AI art platform and NFT marketplace built with Next.js, Tailwind, Ethers, IPFS, and Solidity.

## Designed for Signal

MurMur turns creative intent into on-chain digital artifacts. The project combines AI-assisted image workflows with minting and marketplace primitives so creators can move from idea to collectible in one surface.

## System Topology

- **Frontend:** Next.js App Router + TypeScript UI
- **Styling:** Tailwind CSS
- **Wallet + chain integration:** Ethers v6
- **Asset storage:** Pinata-backed IPFS upload route
- **Smart contracts:** Solidity NFT and marketplace foundation

## From Signal to Structure

1. User connects wallet.
2. User generates artwork via AI workflow.
3. App uploads metadata/assets to IPFS.
4. User mints NFT from the generated artifact.
5. Minted NFT is listed via marketplace foundation flows.

## Operational Layers

- AI artwork generation
- Server-side IPFS upload route
- NFT minting flow
- Marketplace starter logic
- Wallet connection experience

## Why This Exists

Creative output is increasingly AI-native, but ownership and distribution are still fragmented. MurMur exists to unify creation, provenance, and exchange into one coherent product experience.
